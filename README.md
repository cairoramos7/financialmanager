# Gerenciador de finanças pessoais com PHP OO (financial-manager)

Gerenciador de finanças pessoais com PHP 7.1 orientado a objetos.

# Features!

  - Orientação a Objetos (POO)
  - Dependency Injection
  - SOLID
  - MVC
  - Routing
  - Template Engine
  - Composer
  - PHP CodeSniffer
  - Bootstrap
